APRESENTAÇÃO - 

NOVA BELO - É UM SOFTWARE PARA DESKTOP QUE TEM POR FINALIDADE ORGANIZAR, OTIMIZAR E AUXIARLAR AUTO ESCOLAS.

DESCRIÇÃO DO PROJETO - 

ESSE SOFTWARE BUSCA, AMARZENAR DADOS DOS CLIENTES CADATRADADOS, APRESENTAR PACOTES DE SERVIÇOS, FAZER FLUXO DE CAIXA ENTRADA E SAÍDA, ASSIM COMO FECHAMENTO MENSAL 
E RECEBER OS DADOS DO PROCESSO DE CADA CLIENTE, FORNECER INFORMAÇÕES COMO ANIVERSARIANTES DO DIA, VENCIMENTOS DE PROCESSO E EXAMES. 

DESCRIÇÃO DOS USUÁRIOS -

OS USUÁRIOS DO SISTEMAS SERÃO:

AS ATENDENTES - TERÃO SENHAS ESPECIFICAS PARA LANÇAMENTOS DE INFORMAÇÕES, ALTERAÇÃO DE CADASTRO DE ALUNOS ENTRE OUTRAS FUNÇÕES DE ALTERAÇÃO E INCLUSÃO DE DADOS.
OS INSTRUTORES - TERÃO SENHAS ESPECIFICAS PARA CONSULTA DE QUADRO DE AULAS E EXAMES - (NÃO PODENDO FAZER QUALQUER TIPO DE ALTERAÇÃO, INCLUSÃO OU EXCLUSÃO DE DADOS).
OS DIRETORES - TERÃO SENHAS ESPECIFICAS PARA TODAS AS FUNÇÕES EXECETO ALTERAÇÃO DO CAIXA. 
O DONO - TERAR UMA SENHA MESTRE COM TODAS AS FUNÇÕES.  

NECESSIDADES OBSERVADAS E REGRAS DE NEGÓCIO
O CLIENTE PRECISA DE PROGRAMA QUE ATENDA AS NECESSIDADES DE CADASTRO DE ALUNOS DE SUA AUTO ESCOLA, ONDE CADA FUNCIONÁRIO TENHA PERMISSÕES ESPECÍFICAS PARA UTILIZAR O PROGRAMA.
TAMBÉM É NECESSÁRIO QUE O PROGRAMA ARMAZENE INFORMAÇÕES DE PAGEMENTOS, ENTRADAS E SAÍDAS DE CAIXA. SE TRATAR DE UMA AUTO ESCOLA QUE OFERECE SERVIÇOS DE AULAS PRÁTICAS, HABILITAÇÕES
PARA TODAS AS CATEGORIAS A, B, C, D e E, ASSIM COMO SERVIÇOS DE RENOVAÇÃO, LICENCIAMENTO, TRANSFERÊNCIA, REGISTRO DE CNH E PONTUAÇÃO. O MAIS URGENTE É O CADASTRO DE ALUNOS E CLIENTES
E INFORMAÇOES DE FLUXO DE CAIXA.
